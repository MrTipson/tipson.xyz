# Otzdarva recommended killer builds
Static webpage that is generated from Otzdarva's killer builds spreadsheet.

Image links, names and descriptions are scraped from the Dead by Daylight [wiki](https://deadbydaylight.fandom.com).

Everything is integrated via GitHub actions and hosted on GitHub pages ([link](https://mrtipson.github.io/otz-builds)).
